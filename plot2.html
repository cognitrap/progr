Att rita grafer (plot)¶

De frslag p hur man ritar grafer som framfrs hr grundar sig p matplotlib.pyplot. Det kallas att plotta med Pyplot. Det ger snabbt resultat. Om man vill kontrollera ritandet mer i detalj s rekommenderar python.org att man anvnder det objekt orienterade skrivsttet i stllet. Vi behandlar inte det hr utan det kommer att finnas p en annan plats.

Hr p jupyter ritas graferna p ett visst stt med de instllningar som finns. Ritar du dem p replit.com eller p din egen dator kommer det att se annorlunda ut.

Mallen¶

Fr att kunna rita grafer mste vi importera moduler. Vi har konstaterat att vi i grundutfrandet inte kan gra sin(1.2). Men om vi importerade math-modulen s gick det bra. Men fr att kunna rita nskar vi oss mer. Fr att kunna rita fr hand behver vi en vrdetabell och det behver python ocks. S vi mste ha en lista med x-vrden tex. Lx=[0.1,0.2,0.3,0.4]. Detta kan range() hjlpa oss med. Men sedan vill vi berkna sinusvrden fr alla talen i listan. D kan man gra en loop med sin(Lx[i]). Men man har fixat detta problem och lagt det i en modul s man kan direkt skriva sinus av en lista och f sinus fr alla elementen i listan. Kommandot sin(Lx) r tilltet och python gr en ny lista med alla sinusvrdena. Vi behver en modul som heter numpy (numerical python). Vi testar detta kommando frst innan vi behandlar hur vrdetabellen ska ritas.

Man skulle kunna sga att det fungerar s hr:

$$ \begin{align*} \sin\left(Lx\right) & =\sin\left(\left[0.1,0.2,0.3,0.4\right]\right)=\left[\sin\left(0.1\right),\,\sin\left(0.2\right),\,\sin\left(0.3\right),\,\sin\left(0.4\right)\right]=\\ & \left[0.0998,\,0.1987,\,0.2955,\,0.3894\right]=Ly \end{align*} $$

Man har gjort det nnu smidigare genom ett kommando man infrt. De heter linspace(). linspace skapar en datatyp som heter array (anvnds fr vektorer). array har inte samma egenskaper som en lista. Skillnaden syns om man berknar tex. 2*L dr L r en lista respektive en array. Om L r en lista gller konkatenering. Om L r en array gller att varje element multipliceras med 2.

Skilj p linspace och lista.

Utsmyckning¶

Naturligtvis kan man sjlv ndra koordinatsystem, rita flera kurvor i samma diagram, ndra frger p kurvorna, lgga in frklarande text osv.

Exempel fljer. Observera att i anropet p funktionen plt.plot() anvnds nyckelord, vilket vi behandlat i 2.4 om funktioner.

In [ ]:
import matplotlib.pyplot as plt
import numpy as np

# Vrdetabell
x = np.linspace(-2*np.pi, 2*np.pi, 20)
y = np.sin(x)

# Rita med utsmyckning
plt.plot(x,y, color="green",
         linestyle='dashed', 
         marker='o', 
         markerfacecolor='blue', 
         markersize=12, 
         linewidth=2)

plt.title('En sinusfunktion')

plt.show()

Observera hur hela kommandot plot() r uppdelat p flera olika rader, det kar lsbarheten. Det r tilltet att gra s och python vntar p den avslutande hgerparentesen.

  • color="green" betyder att kurvan ritas med grn frg
  • linestyle="dashed" betyder att kurvan r streckad
  • marker="o" innebr att datapunkterna markeras med en cirkelskiva
  • markerfacecolor="blue" gr att cirklarna r fyllda med bltt (inte grna som kurvan)
  • markersize=12 anger storleken p markren
  • linewidth=6 anger bredden p kurvan

Som du kanske inser frn tidigare avsnitt s har flera av parametrarna (color, linestyle osv) egna default (frinstllda) vrden som anvnds om du inte anger ngot. Vi har tex. anget att color ska ha vrdet "green", men om vi inte skickar argumentet "green" s finns det ett frinstllt vrde som python anvnder.

Experimentera. Observera stavningen av frg, det r color inte colour. Observera ocks att det r strngar som ska anges s citattecken mste vara med.

Staplade grafer¶

Fr att kunna rita s kallade subplots behvs en styrning av var graferna ska placeras. Man tnker i rader och kolumner. Kommandot r subplot(nrows, ncols, index). nrows anger antalet rader och ncols anger antalet kolumner. Index anger var grafen ska ritas. Index gr frn vnster till hger och uppifrn och ner.

import matplotlib.pyplot as plt

Teckenförklaring (legend)¶

Om flera kurvor ritas kan det vara bra att ge information om dem. I detta fall lter vi python bestmma var det finns plats. Kommandona som anvnds r:

plot(x,y, label='abs(x)') , plot(x,y,'k:', label='exp(-x)*sin(x)') Dvs. vi anger en etikett eller label() fr att kunna sra p kurvorna.

Teckenfrklaringen: legend(loc='best', fontsize='small')

In [ ]:
import numpy as np
import matplotlib.pyplot as plt

# abs(x)
x = np.linspace(-4, 4, 100)
y = abs(x)
plt.plot(x,y, label='abs(x)')  # Denna kurva fr etiketten abs(x)


# exp(-x**2)*sin(x)
x = np.linspace(-4, 4, 100)
y = np.exp(-x**2)*np.sin(x)+2
plt.plot(x,y,'k:', label='exp(-x**2)*sin(x)+2')  # svart (black) och pricklinje (dotted) linje


plt.legend(loc='best', fontsize='medium')

plt.show()

Kommentar angende plt.hist(x, [0,2,4,6,8,10])

Argumentet bestr av 2 listor. Datamngden ska finnas i listan x. Listan efter x r listan som anger var grnserna fr de olika binen ska g, de olika grupperna. Intervallet from. 0 till 2 r frsta binet; allts inklusive 0 och exklusive 2. Fr det sista intervallet 8 till 10 gller dock inklusive fr bda grnserna. Vi ser frn datamngden att det finns inga 0:or och 5 stycken 1:or vilket ger en stapel med hjden 5 fr intervallet 0 till 2; vi ser vidare att det finns 5 st 2:or och 2 st 3:or s 7 blir hjden fr intervallet 2 till 4.

Vi kan ocks bara ange antalet intervall vi nskar och lmna resten till python som d konstruerar intervall som r lika stora. Anger vi inget intervall s r det frinstllt 10 stycken.
